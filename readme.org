* Prerequisites
- premake4
- GNU make
- oolua
- lua
- physfs
- zlib
- pthread
- glew
- sfml 2.0
- libGL
- Clang 3.1+ or GCC 4.7+
* Compilaton
- enter the folder with premake4.lua
- run "premake4 gmake"
- run make
* Running
- The binary will be compiled as topaz_test
- make sure topaz_test is in a folder containing the subfolders:
  - textures
  - animations
  - models
